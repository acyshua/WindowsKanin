# WindowsKanin
A repository for my collection of configurations, wallpapers, and applications that can be used to rice Windows 10/11

This repository serves as a backup just in case something happens to my files.
# Preview:
![alttext](https://github.com/acyshua/WindowsKanin/blob/main/preview.png)
# Stuff used:

Wallpapers: wallpaper-a-day.com and Mifuru (Pixiv) (https://www.pixiv.net/en/users/488766)

Taskbar: NiceTaskbar (Windows 10, Microsoft Store, for Windows 11 use Translucent TB)

Status Bar: YASB (https://github.com/amnweb/yasb)

File Searcher: Flow Launcher (https://github.com/flow-launcher/flow.launcher)

Terminal: Windows Terminal (https://github.com/microsoft/terminal)

Window Manager: komorebi (https://github.com/LGUG2Z/komorebi)

CAVA (https://github.com/karlstav/cava)

fastfetch (https://github.com/fastfetch-cli/fastfetch)

pipes-rs (https://github.com/lhvy/pipes-rs)

WSL (https://github.com/microsoft/WSL)

tty-clock (https://github.com/xorg62/tty-clock)

The configurations are not created by me:
